# Spring Boot Orders Management Streams

## Description

Kafka Streams in a Spring Boot 3.3.0 application for learning purposes.

## Technologies

- Java 21
- Gradle
- Spring Boot 3.3.0
- Kafka
- Kafka Streams
- Logback
- Lombok
- Docker Compose

## How to run

### Start Kafka and Zookeeper

```shell
docker-compose up -d
```

### Start the application

```shell
gradle bootRun
```
